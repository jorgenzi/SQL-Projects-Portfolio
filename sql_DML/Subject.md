В этом проекте я научилась извлекать данные из базы с помощью сложных SQL-запросов и изменять их при помощи языка управления данными (DML) — вставлять, обновлять и удалять записи для поддержания актуальности базы.

Умение эффективно извлекать и модифицировать данные из базы поможет при разработке систем учета, анализа продаж, персонализации сервиса, а также при создании отчетов и дашбордов.

Задание 00:
SQL-запрос, который возвращает список:
- названия пицц,
- цены на пиццы,
- названия пиццерий,
- даты посещений

для клиента Kate с ценами в диапазоне от 800 до 1000 рублей.

Результаты упорядочены по названию пиццы, цене и названию пиццерии.

Задание 01:
Найдены все идентификаторы меню, которые никто не заказывал.
Результат отсортирован по идентификатору.

Задание 02:
Найдены пиццерии, которые посещали чаще женщины или чаще мужчины.  
Результат отсортирован по названию пиццерии.  

Задание 03:
Найдено объединение пиццерий, в которые заказы делали исключительно женщины, и тех, в которые заказы делали исключительно мужчины. 
Результат отсортирован по названию пиццерии.  

Задание 04:
SQL-запрос, который вернет список пиццерий, которые посещал Андрей, но из которых он не делал заказов.  
Результат отсортирован по названию пиццерии.  

Задание 05:
Найдены совпадающие названия пицц с одинаковой ценой, но из разных пиццерий. Результат отсортирован по названию пиццы.  
Ниже приведен пример данных.  

Задание 06:
Добавлена новая пицца с названием «greek pizza» (id = 19) по цене 800 рублей в ресторане «Dominos» (pizzeria_id = 2).

Задание 07:
Добавлена новая пицца с названием «sicilian pizza» (с ID, рассчитанным как «максимальный существующий ID + 1») по цене 900 рублей в ресторан «Dominos».

Задание 08:
Зафиксированы новые посещения ресторана Domino's Денисом и Ириной 24 февраля 2022 года.  

Задание 09:
Зарегистрированы новые заказы от Дениса и Ирины 24 февраля 2022 года на новое блюдо меню — «sicilian pizza».  

Задание 10:
Изменена цена на «greek pizza» на 10% от текущего значения.

Задание 11:
Зарегистрированы новые заказы "greek pizza" для всех клиентов 25 февраля 2022 года.

Задание 12:
Написаны два SQL-запроса (DML), которые удаляют все новые заказы из задания 12 по дате заказа. И один SQL-запрос для удаления пиццы «greek pizza» из меню.
