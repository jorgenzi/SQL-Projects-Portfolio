В этом проекте я научилась создавать SQL-запросы с использованием внутренних подзапросов в частях FROM и SELECT, а также фильтровать данные по диапазону дат. Для реализации запросов используется учебная модель базы данных с именем "model.sql"

Я потренировала навыки работы с условиями выборки, сортировкой по нескольким полям с разными направлениями и структурированием запросов для получения нужной информации.

Задание 00:
Запрос SELECT, который возвращает имена и возраст всех людей из города «Казань».

Задание 01:
Запрос SELECT, который выводит имена и возраст всех женщин из города «Казань». Результат отсортирован по имени.

Задание 02: 
Запрос SELECT, который вернёт список пиццерий (название и рейтинг) с рейтингом от 3.5 до 5 включительно, отсортированный по рейтингу.

Задание 03: 
Запрос SELECT, который возвращает уникальные идентификаторы людей (без дубликатов), посетивших пиццерии в период с 6 по 9 января 2022 года (включительно), либо посетивших пиццерии с идентификатором 2.
Результат отсортирован по идентификатору человека в порядке убывания.

Задание 04:
SQL-запрос, который возвращает все столбцы из таблицы person_order, где идентификатор является четным числом.
Результат отсортирован по этому идентификатору.

Задание 05: 
Запрос SELECT, который возвращает имена людей и названия пиццерий на основе таблицы person_visits с датами посещений в период с 7 по 9 января 2022 года (включительно), используя внутренний запрос в части FROM.
Сортировка по имени человека в порядке возрастания и по названию пиццерии в порядке убывания.
